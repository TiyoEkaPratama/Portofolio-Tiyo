Ini adalah langkah penting dalam perjalanan karir saya di dunia web development.

UI - UX DESGIN

By: Admin 999 999

Saya saat ini sedang terlibat dalam proyek yang sangat menarik, yaitu merancang dan mengembangkan website portfolio dengan fokus pada pengalaman pengguna (UX) dan tampilan antarmuka (UI) yang intuitif serta menarik. Dalam proyek ini, saya memprioritaskan tampilan yang modern namun tetap sederhana, memastikan setiap elemen mudah diakses dan dipahami oleh pengguna.

Untuk merancang UI/UX, saya menggunakan Figma, sebuah platform desain kolaboratif yang memungkinkan saya untuk bekerja dengan lebih efisien. Dengan Figma, saya dapat membuat prototipe interaktif, melakukan testing desain, dan berkolaborasi secara real-time dengan tim atau klien. Penggunaan Figma tidak hanya memudahkan dalam visualisasi desain, tetapi juga membantu saya untuk merancang pengalaman pengguna yang halus, konsisten, dan responsif di berbagai perangkat.

Website ini nantinya akan menampilkan berbagai aspek dari perjalanan karier saya, mulai dari deskripsi diri, keterampilan yang saya kuasai, hingga proyek-proyek yang telah saya selesaikan. Dengan fokus pada desain yang bersih dan fungsional, portfolio ini akan menjadi representasi visual dari kemampuan saya dalam bidang desain dan pengembangan website.

Anggota KPPS

By: Admin 999 999

Pada Pemilihan Umum Presiden (Pilpres) 2024, saya memiliki kehormatan untuk bergabung sebagai anggota Kelompok Penyelenggara Pemungutan Suara (KPPS). Tugas ini adalah tanggung jawab besar yang sangat penting dalam menjaga demokrasi, di mana saya bersama tim bertanggung jawab untuk memastikan proses pemungutan dan penghitungan suara berjalan dengan lancar, transparan, dan adil di tingkat Tempat Pemungutan Suara (TPS).

Dalam peran ini, saya akan berperan aktif dalam memfasilitasi hak-hak pemilih agar dapat menggunakan hak suaranya dengan nyaman dan aman. Tugas saya mencakup persiapan TPS, pendistribusian logistik pemilu, hingga memastikan seluruh prosedur pemungutan suara mengikuti standar yang telah ditetapkan oleh Komisi Pemilihan Umum (KPU). Selain itu, sebagai anggota KPPS, saya juga bertanggung jawab untuk mengawasi proses penghitungan suara secara transparan serta melaporkan hasilnya sesuai aturan.

Menjadi bagian dari KPPS bukan hanya sebuah pekerjaan teknis, tetapi juga sebuah dedikasi untuk memperkuat pilar-pilar demokrasi di Indonesia. Saya siap memberikan kontribusi terbaik untuk memastikan Pilpres 2024 berlangsung dengan sukses, dan bahwa suara setiap warga negara dihitung dengan jujur dan adil.
